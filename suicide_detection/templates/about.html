About This Project

Academic B.Tech Project-I Report · CGC College of Engineering, Landran

Abstract

Suicide is a serious global public health concern with approximately 700,000 lives lost annually (WHO, 2021). This project proposes an automated system for detecting suicidal ideation from social media textual data using Natural Language Processing (NLP) and deep learning techniques.

The system analyzes posts and classifies them as suicidal or non-suicidal based on linguistic patterns, emotional indicators, and contextual information. The methodology includes five stages: data collection, preprocessing, TF-IDF and GloVe feature extraction, Logistic Regression and LSTM model development, and evaluation using accuracy, precision, recall, and F1-score.

Objectives
  • Collect and preprocess textual data from publicly available suicide ideation datasets
  • Implement a Logistic Regression baseline model using TF-IDF feature extraction
  • Develop an LSTM deep learning model using GloVe word embeddings
  • Evaluate and compare models using accuracy, precision, recall, F1-score
  • Demonstrate practical AI application for mental health monitoring
  • Document ethical AI framework for responsible deployment
